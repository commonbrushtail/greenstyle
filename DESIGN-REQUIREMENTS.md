# Green Style Website - Design Requirements

## Brand Identity

### Color Palette
- **Primary**: Green tones (eco-friendly, natural, sustainable)
  - Suggested: #2D5016, #4A7C2E, #6B9F4A
- **Secondary**: Earth tones (brown, beige)
  - Suggested: #8B7355, #D4C5B0
- **Accent**: Fresh, vibrant colors
  - Suggested: #87C540, #FFA500 (for CTAs)
- **Neutrals**: White, light grey, dark grey
  - Background: #F8F9FA
  - Text: #333333

### Typography
- **Headings**: Modern, clean sans-serif (Thai-compatible)
  - Suggested: Prompt, Kanit, Sarabun (Google Fonts - Thai support)
- **Body**: Easy-to-read, professional
  - Suggested: Sarabun, Noto Sans Thai
- **Font sizes**: Responsive scaling for mobile

## Responsive Design
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px
- Ensure touch-friendly buttons on mobile
- Hamburger menu for mobile

## Accessibility
- WCAG 2.1 AA compliance
- Alt text for all images
- Proper heading hierarchy
- Keyboard navigation
- Sufficient color contrast

## Performance
- Page load time < 3 seconds
- Optimized images (WebP format)
- Lazy loading for images
- Minified CSS/JS
