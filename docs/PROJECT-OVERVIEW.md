# Green Style Website - Project Overview

## Project Goal
Redesign and develop a modern website for Green Style Co., Ltd., an environmental consulting and eco-products company in Thailand.

## Target Audience
- Businesses seeking environmental consulting (CFO, CFP services)
- Organizations looking for sustainability training
- Consumers interested in eco-friendly products
- B2B and B2C customers

## Key Objectives
1. Showcase environmental consulting services (Carbon Footprint calculations)
2. Promote training courses on sustainability
3. Display eco-friendly product catalog
4. Build trust and credibility in the environmental sector
5. Generate leads for consulting services

## Technology Stack
- **Framework**: Next.js 14+ (React)
- **CMS**: Sanity.io (Headless CMS)
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Deployment**: Vercel (recommended for Next.js)
- **Content Language**: Thai (primary), English (secondary)

## Project Phases
1. **Phase 1**: Content organization and sitemap
2. **Phase 2**: Design mockups and wireframes
3. **Phase 3**: Frontend development
4. **Phase 4**: Backend development (if needed)
5. **Phase 5**: Content migration
6. **Phase 6**: Testing and launch
